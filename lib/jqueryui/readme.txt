To add the JQUERY UI, you must add the following lines in the head section:
	<link rel="stylesheet" href="http://ajax.googleapis.com/ajax/libs/jqueryui/1.8.21/themes/base/jquery-ui.css" type="text/css" media="all" />

	<script language="javascript" type="text/javascript" src="../jquery/js/jquery-1.7.2.min.js"></script>
	<script language="javascript" type="text/javascript" src="../jqueryui/js/jquery-ui-1.8.21.custom.min.js"></script>
